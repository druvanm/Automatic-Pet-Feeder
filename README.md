# IoT-Based Automatic Pet Feeder

## 📌 Overview
This project is an IoT-based Automatic Pet Feeder developed using ESP8266, Servo Motor, and the Blynk IoT platform. The system allows users to feed their pets remotely through the Blynk mobile application. A servo motor dispenses food, and a buzzer provides an audible indication during feeding.

## ✨ Features
- Remote pet feeding using the Blynk app
- Servo motor-controlled food dispensing
- Buzzer indication during feeding
- Wi-Fi connectivity using ESP8266
- Simple and user-friendly design

## 🛠️ Components Used
- ESP8266 NodeMCU
- Servo Motor
- Buzzer
- Pet Food Container
- Jumper Wires
- Breadboard
- Blynk IoT Platform

## 💻 Software Used
- Arduino IDE
- Blynk IoT
- ESP8266 Board Package

## 📂 Project Files
- `Automatic_Pet_Feeder.ino` – Source code
- `Automatic-Pet-Feeder.pdf` – Project documentation

## 🚀 Working
1. Connect the ESP8266 to Wi-Fi.
2. Open the Blynk mobile application.
3. Turn ON the switch widget.
4. The servo rotates to dispense food.
5. The buzzer sounds during operation.
6. Turning OFF the switch stops the feeder.
